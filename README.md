# DESFIO CLARKE ENERGIA

O presente projeto é uma API REST desenvolvida em Python com FastAPI para auxiliar na escolha da melhor tarifa de energia elétrica para clientes do mercado livre de energia.

## Tecnologias utilizadas 

### Backend

- Python 3.12
- FastAPI
- Alembic
- SQLModel
- Pytest

### Frontend

- React 19
- Vite
- Tailwind CSS
